INTRODUCCIÓN A DISEÑO WEB



¿Qué es HTML?

Es la abreviatura de HyperText Markup Lenguaje, y es el lenguaje que todos los programas navegadores usan para presentar información en la WORD WIDE WEB (w.w.w.). Es sencillo y se basa en el uso de etiquetas.



¿Qué nos podrian incluir y mencionar las etiquetas?

Podrian incluir una serie de atributos o parámetros, mayormente opcional. Y nos permiten definir diferentes posibilidades a características de la misma.



¿Qué se necesita para crear un documento HTML?

Un procesador de texto para escribir y editar el código HTML. Este debe ser basado en el código ASCII. Y un navegador web cualquiera.



Secuencia para construir un documento nuevo

  • Crear el documento HTML

  • Guardar como "<nombre>".html o .htm

  • Abrir documento con navegador



Las reglas de formato

El espacio en blanco es ignorado: Ya que un documento HTML, puede estar en cualquier fuente y la ventana del navegador puede ser de cualquier tamaño.

Las etiquetas pueden ser escritas en mayús o minus: Se aconseja que sea en mayúsculas para distnguirlas del texto normal.

Existe normalmente una etiqueta de inicio, y una de fin: Contendrán el mismo texto, pero la del final añadirle una barra "/". No obstante, existen algunas que no necesitan cierre. Ejemplos: <P>; <BR>; <IMG>.



Estructura básica de un documento HTML

<HTML>"inicio del documento".


<HEAD>"Inicio de cabecera". </HEAD>


<TITLE>"Título". </TITLE> "Inicio y Fin del titulo"


<HEAD>"Fin de cabecera". </HEAD>


<BODY>"Inicio del cuerpo"


"Instrucciones"


</BODY> "Fin del cuerpo"


</HTML>




Etiquetas más utilizadas
Etiqueta Descripción Ejemplo
<a> Enlace a otra página o sección <a href="pagina.html">Ir a página</a>
<p> Define un párrafo <p>Texto de un párrafo</p>
<h1> al <h6> Encabezados de diferentes niveles <h1>Título principal</h1>
<div> Contenedor genérico <div>Contenido aquí</div>
<header> Encabezado de sección <header>Encabezado del sitio</header>
<nav> Menú de navegación <nav>Enlace de menú</nav>
<footer> Pie de página <footer>Pie de la página</footer>
